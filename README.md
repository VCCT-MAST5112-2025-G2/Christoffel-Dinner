## Christoffel Dinner App

Cheff christoffel dinner app is an app where the general user would be able to use to order various types of pasts meals that the establisment provides by the cheff themself 
The application allows the user to view the menu and allows the user to make food orders and pay within the application

## Features

Features within the application consist of: 

•	Homepage 
•	Menus page 
•	Cheff chistoffel page (Admin)


# Homepage

-	The home page serves as the start of the application. This page is where the user will first see when they open the application 
-	The home page consists of the Dinners logo, a welcome message to the user and a button where the user can click to view the menu of the dinner

# Menus Page 

-  The Menue page consist of a navigation ( Starters, Mains and Deserts).
-  The menue page is where the users would be able to view the various menues the chef imputs
-  The menue items would include the image, name, discriptions and price.

# cheff christoffel Page ( Admin Page) 

- The Admin Page is a page specialy designed for chedd christoffel
- This page serves as a page where cheff can imput menue items he specialises/ servesive on
- The admin page allows the owner to import images, give discription of the meal and give the item name and price and would be select which genera the iteam falls into ( starter, main, Desert)
- Then onces added it would refelect in the menues page.
